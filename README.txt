== Description ==


This plugins is for adding a shortcode to Wordpress and then you can user the shortcode [ember language="fr"] to insert our app. The app should be in the plugin folder under ember-apps. 


== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `ember-app.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Use the shortcode [ember language="fr"]
